Public Demo Page

Soltistix

Soltistix is a demo concert ticketing and resale platform built for academic and sandbox testing use. The platform supports concert browsing, ticket inventory, resale listings, payments, refunds, and seller payout demonstrations in test environments.

This public page is provided for platform verification and support reference during third-party sandbox setup.

Platform Scope

Concert discovery, primary purchase, resale listing and resale purchase orchestration.

Environment

Demo and sandbox usage only. No live financial activity is intended on this page.
